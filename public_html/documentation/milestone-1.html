Personal Website Project

Purpose:

My PWP will be an expansion of an existing portfolio site that shows a sampling of design work and some descriptive text, then adding programming skills and changing the focus to one of a competent web developer with strong front-end abilities.

Audience:

The audience for this project will be potential employers and customers. Employers in that I will be looking for full-time employment, Customers in that I've always taken a freelance job here and there when I have the time.

Goals:

The goals of this project will be to attract employers and get job offer(s), and to keep as a high-quality repository for visual design work (in addition to GitHub for coding work).

Persona:
Mike Smith

Mike is the IT manager of a medium-sized company. He has been tasked with creating a brand-new website to be deployed with the visuals from an ad-agency-conceived branding package that features a new customer focus. The current website uses PHP/MySql with a mix of dynamic pages and old static pages that were built with DreamWeaver, but the new site will use an industry-standard, modern toolchain.

Mike uses his desktop work computer with its 4-monitor setup for almost all of his work-related tasks, as he works long hours and likes to leave his work at the office when he leaves for the evening. Otherwise he uses an iPhone 12 Pro Max for occasional browsing and out-of-office email retrieval. While he uses all available browsers for compatibility testing, he favors Firefox for general browsing. His company subscribes to Microsoft 365 suite, including Outlook for email.

Frustrations:

The inconsistency of technology within the website has been a long-standing frustration, so Mike's boss sees this as an opportunity for a brand-new website that will be revealed as an integral part of the new branding release. It will use JavaScript exclusively, and Bootstrap for styling look/feel. Unfortunately, his current staff is somewhat unfamiliar with the application of these new techniques, so he is looking for someone with these skills to flatten the learning curve, and get this job finished inside the deadline he has been provided.

User Story:

Mike receives daily push notifications from services like Indeed and ZipRecruiter for possible matches to an employee profile he filled out seeking a candidate with specific skills. My name appears in both emails, so he decides to visit my portfolio site.

Goals:

The goals of my personal portfolio site will be to present specific skills that Mike is looking for, and to let him know that I come with a host of related skills that most developers don't, which is ad agency, branding, and visual design experience. Having this additional experience will hopefully prove to be desirable to Mike, as I already understand branding and agency culture, so it will be easy for me to deliver product that conforms to the agency's style guide.
